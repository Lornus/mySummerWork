# Attends in automation 
Branch with tests (mocha + WebdriverIO + Chai + Allure HTML reports)

Site under testing - https://npplanets.herokuapp.com/ 

# Launching tests 

Install [nodeJS](https://nodejs.org/en/download/) if you don't have.

```
npm install
```
For running tests
```
npm test
```
For getting report
```
npm run report 
```
